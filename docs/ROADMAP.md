# 로드맵

## MVP 범위
- 정적 JSON 데이터로 3~5개 뮤지엄 락커 렌더링
- 그리드/토픽/맵 레이아웃 전환 버튼
- 문 열림 + 내부 확장 + 콘텐츠 등장 애니메이션
- 관계 곡선과 텍스트 슬라이스 기반 노출
- 해시 기반 워프 기능 (NFC URL 대응)

## 추가 구현 아이디어
- 실제 드래그 인터랙션 연결 및 탄성 이동
- SVG 줌/팬 컨트롤, 미니맵 추가
- 검색/필터(지역, 주제) 기능
- 이미지 갤러리 캐러셀 및 비디오 삽입
- WebGL 혹은 Canvas 기반 고성능 렌더링으로 확장

## 테스트 및 개선 계획
- 주요 브라우저(Chrome, Safari, Edge)에서 성능 및 터치 제스처 확인
- 모바일 NFC 시나리오별 URL 워프 확인
- Lighthouse로 접근성·성능 점검 후 토큰/컬러 대비 튜닝
- 애니메이션 프레임 드랍 체크 및 레이아웃 알고리즘 최적화
