# 프로젝트 개요

## 기획 의도
- 네덜란드의 주요 뮤지엄을 "수장고 사물함" 메타포로 재구성하여, 물리적 조작감을 주는 웹 인포그래픽.
- 방문자는 문을 열거나 끌어당기며 관계를 체험하고, NFC 스티커를 통해 현장 경험을 디지털로 확장.

## 사물함 메타포
- 각 뮤지엄은 하나의 사물함 문으로 표현되며, 문이 열리면 내부에 상세 배경이 확장되고 콘텐츠가 나타난다.
- 서로 붙어 있던 문이 떨어지면 탄성 있는 곡선(치즈 스트레치)이 생성되어 협업·교류 같은 관계를 보여준다.

## 사용자 인터랙션 흐름
1. 초기 진입: 빽빽한 그리드 형태의 사물함이 보이고, FAB로 소개/갤러리를 열 수 있다.
2. 문 클릭: 왼쪽 힌지 축을 기준으로 문이 열리고, 내부 배경이 확장되며 상세 정보가 페이드인.
3. 드래그: 사물함을 끌어 서로를 떨어뜨리면 곡선이 늘어나고 텍스트가 한 글자씩 드러난다.
4. 해시/NFC: `/#museum-id`로 접근 시 해당 사물함이 자동으로 강조·오픈된다.
5. 레이아웃 토글: Grid / Topic / Map 모드 간 전환이 애니메이션으로 이어지며 위치가 재배치된다.
