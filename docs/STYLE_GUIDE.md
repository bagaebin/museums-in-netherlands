# 스타일 가이드

## 컬러 팔레트
- 배경: `#0f172a`
- 패널: `#1e293b`
- 하이라이트 1: `#22d3ee`
- 하이라이트 2: `#f97316`
- 텍스트 기본: `#e2e8f0`
- 서브 텍스트: `#94a3b8`

## 타이포그래피
- 기본 폰트: Inter, system-ui
- 제목: 18~24px, 굵기 700~800
- 본문: 14~16px, 굵기 400~500, line-height 1.5
- 캡션/라벨: 12~13px, 대문자 또는 트래킹 +0.4px

## 사물함 스타일 규칙
- 코너 radius: 12~16px
- 외곽선: 1~2px, `rgba(255,255,255,0.12)`
- 그림자: `0 12px 24px rgba(0,0,0,0.3)` 기본, 강조 시 추가 광채 `rgba(34,211,238,0.6)`
- 클립 패스: inset 또는 circle을 사용해 배경 확장
- 배경 그라디언트: 어두운 베이스 + 청록/주황 포인트 그라디언트
