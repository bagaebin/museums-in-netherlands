# 스타일 가이드

## 컬러 팔레트
- 배경: `#0c0c0f`
- 패널: `#11111a`
- 포커스/하이라이트: `#ffb347`
- 액센트 강: `#ff8c42`
- 텍스트 기본: `#f6f6fa`
- 보조 텍스트: `#9fa3b4`

## 타이포그래피
- 기본 폰트: Inter / Pretendard / system-ui
- 제목: 굵게, 자간 약간 증가
- 본문: 14–16px, 가독성 높은 대비 유지
- 캡션/레이블: 12px, 대문자 및 트래킹 확장 가능

## 형태 규칙
- 사물함 모서리 radius: 12–16px
- 문 테두리 두께: 1px (#2c2d38)
- 그림자: `0 10px 30px rgba(0,0,0,0.35)` 정도로 깊이감 부여
- clip-path 확장 시 둥근 모서리 유지(16px)
