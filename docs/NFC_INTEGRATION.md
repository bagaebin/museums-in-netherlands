# NFC 연동

## URL 기록 방식
- 각 NFC 스티커에는 `https://site.com/#museum-id` 형태의 URL을 기록한다.
- `id`는 `data/museums.json`의 `id` 필드와 동일해야 한다.

## 워프 로직
- 페이지 로드 시 `window.location.hash`를 읽어 대상 ID를 추출한다.
- 매칭되는 사물함이 있으면 자동으로:
  1) 해당 사물함을 `open` 상태로 전환
  2) 강조 애니메이션(점선 링) 적용
  3) DOM 스크롤/뷰포트 중앙 정렬 시도

## 플랫폼 동작 예상
- **안드로이드**: Web NFC 또는 브라우저에서 URL을 바로 열어 해시가 전달됨.
- **iOS**: 브라우저 실행 후 동일하게 URL 해시가 포함되어 동작.
- 네트워크 오프라인이어도 이미 열린 페이지라면 해시 변경만으로 워프 가능.
