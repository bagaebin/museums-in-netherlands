# NFC 연동

## URL 기록 방식
- NFC 스티커에 `https://도메인/#museum-id` 형태의 URL을 기록한다.
- 브라우저에서 태그를 스캔하면 해당 URL로 이동해 해시를 포함한다.

## 해시 기반 워프
- 페이지 로드 시 `window.location.hash`를 읽어 ID를 추출한다.
- 해당 ID에 매칭되는 락커를 찾아 뷰포트를 스크롤·센터링하고, 자동으로 문을 연다.
- 추가로 링 애니메이션 등 강조 효과를 주어 태그 스캔 피드백을 제공한다.

## 예상 동작 (Android / iOS)
- **Android (Web NFC)**: 크롬에서 URL이 바로 열리며, PWA 상태에서도 동일하게 작동한다.
- **iOS**: iOS의 NFC 리더기는 사파리를 통해 URL을 연다. 해시가 포함되어 있기 때문에 동일하게 워프 로직이 동작한다.
- 네트워크가 없어도 설치된 앱 캐시(서비스 워커 적용 시)로 빠르게 열 수 있다.
