# 프로젝트 개요

네덜란드 전역의 박물관을 "사물함 벽" 메타포로 시각화한 인터랙티브 인포그래픽 웹사이트입니다. 사용자는 빽빽하게 정렬된 사물함 문을 드래그하거나 열어보며 수장고를 직접 만지는 듯한 경험을 얻게 됩니다. NFC 태그를 스캔하면 해당 박물관의 문이 바로 열리며, 물리적-디지털 연결감을 강화합니다.

## 기획 의도
- 박물관 간 협업, 주제, 지리 정보를 한 화면에서 탐험할 수 있도록 직관적인 벡터 타일로 표현
- 문을 열고 내부 공간이 확장되는 애니메이션을 통해 "보관 → 공개"의 흐름을 시각화
- NFC 스티커와 웹을 연결해 전시 현장에서 곧바로 디지털 정보로 워프하는 경험 제공

## 핵심 사용자 흐름
1. **레이아웃 선택**: 그리드/토픽/지도 모드 중 하나를 택해 사물함 배치를 바꾼다.
2. **사물함 조작**: 타일을 드래그해 관계 곡선을 늘이고, 클릭 시 문이 힌지 회전하며 열림.
3. **내부 확장**: 문 뒤의 사각형 또는 원형 배경이 clip-path로 확장되어 상세 화면을 형성.
4. **콘텐츠 탐색**: 이미지, 설명, 외부 링크 등 상세 정보를 읽는다.
5. **NFC 워프**: 해시(#museum-id)가 포함된 URL을 통해 특정 사물함을 바로 열고 하이라이트한다.
