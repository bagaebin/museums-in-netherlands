# 프로젝트 개요

이 프로젝트는 네덜란드의 주요 뮤지엄을 사물함(락커) 형태의 인터랙티브 인포그래픽으로 표현한다. 사용자는 촘촘하게 배치된 문을 직접 끌어당기고 열며, 물리적으로 수장고 벽을 만지는 듯한 경험을 얻게 된다.

## 기획 의도
- 관람객이 수장고를 탐색하는 손맛을 웹에서도 재현한다.
- 사물함 문을 열면 내부 공간이 확장되어 상세 페이지가 되는 시각적 전환을 강조한다.
- 관계가 있는 뮤지엄끼리 떨어지면 치즈처럼 늘어나는 곡선을 통해 네트워크를 인지적으로 전달한다.

## 사물함 메타포
- 각 뮤지엄은 하나의 문이며, 문짝은 힌지로 회전한다.
- 문 뒤에는 확장 가능한 내부 공간이 자리잡고 있어 세부 정보를 담는다.
- 촘촘한 격자로 쌓인 문들은 실제 저장 벽을 떠올리게 한다.

## 주요 인터랙션 흐름
1. 기본 그리드에서 뮤지엄 문을 클릭하면 문이 좌측 힌지를 기준으로 열리고, 내부 배경이 확장되며 콘텐츠가 등장한다.
2. 두 문을 드래그로 멀리 떼어놓으면 곡선이 생성되고, 곡선 길이에 따라 관계 라벨이 한 글자씩 드러난다.
3. 레이아웃 토글을 통해 주제별(Topic) 클러스터, 지도(Map) 뷰로 전환하며, 위치 변화가 부드럽게 애니메이션된다.
4. NFC 스티커를 스캔해 특정 뮤지엄의 해시 URL로 이동하면 해당 문이 자동으로 강조·오픈된다.
