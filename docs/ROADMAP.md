# 로드맵

## MVP 범위
- 3가지 레이아웃 모드 전환(그리드/토픽/지도)
- 사물함 드래그, 문 열림, clip-path 확장, 콘텐츠 페이드인
- 관계 곡선 및 글자 단위 노출
- 해시 기반 NFC 워프
- 기본 정보 팝업(FAB)

## 추가 아이디어
- 줌/팬 제스처, 키보드 네비게이션
- 검색/필터(지역, 주제, 협업 횟수)
- 이미지 갤러리 슬라이드, 비디오 임베드
- 관계 강도에 따른 곡선 굵기/색상 코딩
- WebGL/Canvas로 수많은 노드 성능 개선

## 테스트 및 개선 계획
- 브라우저 호환성(Chrome/Safari/모바일) 체크
- NFC URL 인입 플로우 QA (Android/iOS)
- 접근성: 키보드 포커스 순서, ARIA 라벨링
- 데이터 검증: JSON 스키마 테스트 및 타입 가드 추가 예정
