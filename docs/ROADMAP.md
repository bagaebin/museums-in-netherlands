# 로드맵

## MVP 범위
- Grid/Topic/Map 레이아웃 전환 및 애니메이션.
- Locker 컴포넌트의 문 회전, 내부 배경 확장, 상세 콘텐츠 등장까지 단일 흐름 구현.
- 관계 곡선 및 라벨 렌더링.
- NFC 해시 워프 처리.

## 추가 구현 아이디어
- 줌/팬 제스처 및 모멘텀 스크롤 지원.
- 검색/필터(지역, 주제, 시대)와 하이라이트 애니메이션.
- 실제 지도 SVG 오버레이 + 지역별 그라데이션 배경.
- 사물함 드래그 후 위치 고정 및 저장(로컬 스토리지/백엔드).
- 곡선 텍스트의 마스킹 기반 글자 단위 등장 구현.

## 테스트 및 개선 계획
- 주요 브라우저(Chrome, Safari, Firefox)에서 WebGL/Framer Motion 성능 확인.
- 모바일(안드로이드/iOS)에서 해시 기반 자동 포커스 시나리오 테스트.
- 데이터 JSON 스키마 유효성 검사(TypeScript + 런타임 zod 등) 추가.
- 접근성: 키보드 포커스, ARIA 라벨, 명도 대비 점검.
