# 프로젝트 개요

이 웹앱은 네덜란드 주요 미술관들을 `사물함 문` 형태의 벡터 타일로 표현하여, 실제 수장고를 열고 닫는 듯한 촉각적 경험을 제공한다. 사용자는 문을 드래그하고, 떨어지는 순간 생성되는 탄성 곡선을 통해 기관 간 관계를 직관적으로 파악하며, 문을 클릭하면 내부 공간이 확장되어 상세 페이지로 전환된다.

## 사물함 메타포
- 각 미술관은 하나의 사물함 문으로 표현되며, 빽빽한 격자에 배치된다.
- 문을 여닫는 동작이 곧 탐색 행위로 이어져, 데이터 브라우징이 물리적 조작처럼 느껴진다.
- 관계는 치즈처럼 늘어나는 베지어 곡선으로 시각화되어, 수장고 선반을 옮길 때 드러나는 연결감을 연상시킨다.

## 주요 인터랙션 흐름
1. **레이아웃 전환**: Grid / Topic / Map 세 가지 모드로 배치가 부드럽게 변환된다.
2. **문 클릭**: 왼쪽 힌지 기준으로 회전하며 열리고, 이어서 내부 배경이 클립 경로를 확장하며 전체 상세 영역을 차지한다.
3. **상세 진입**: 배경 확장 후 텍스트·이미지가 서서히 나타나며, 링크를 통해 외부 사이트로 이동할 수 있다.
4. **관계 드래그**: 인접한 문을 벌릴 때 곡선이 탄성을 가지며 늘어나고, 경로 길이에 따라 라벨 텍스트가 점차 드러난다.
5. **NFC 워프**: NFC 태그를 스캔하면 해당 미술관의 해시로 진입하며 자동으로 포커스·오픈된다.
