# NFC 연동 가이드

## URL 기록 방식
- NFC 스티커에 `https://site.com/#museum-id` 형식의 URL을 기록한다.
- `museum-id`는 데이터 JSON의 `id` 필드와 일치해야 한다.

## 워프 동작
1. 브라우저가 열리면 `location.hash`에서 id를 추출한다.
2. 해당 id에 맞는 사물함을 찾고, 뷰박스를 해당 좌표 중심으로 이동한다.
3. `activeId`를 설정해 문 회전, 내부 확장, 콘텐츠 등장까지 자동 실행한다.
4. 필요 시 펄스 테두리 등 하이라이트를 추가해 피드백을 준다.

## 플랫폼별 시나리오
- **안드로이드(Web NFC)**: 브라우저가 직접 URL을 열며 해시가 포함되므로 바로 워프된다.
- **iOS**: NFC 리더기가 사파리/크롬을 열어 동일한 URL로 이동하며, 동일하게 해시 기반 워프가 실행된다.
