# 로드맵

## MVP 범위
- 3~5개 뮤지엄 데이터로 그리드/토픽/맵 모드 전환
- 문 열림 + 내부 배경 확장 + 콘텐츠 페이드인
- 드래그 가능한 사물함과 관계 곡선 렌더링
- 해시 기반 워프(NFC URL) 동작

## 추가 아이디어
- 줌/팬 가능한 SVG 뷰포트
- 검색/필터(지역·주제) UI
- 곡선 굵기/색으로 관계 강도 표현
- 다단계 텍스트 애니메이션 (마스킹 기반)
- 모바일 햅틱/사운드 피드백

## 테스트 및 개선 계획
- 주요 브라우저(Chrome, Safari, Edge)에서 3D transform 및 clip-path 호환성 확인
- 모바일 터치 드래그 감도 조정, 성능 프로파일링
- NFC 태그 실기기 검증: 안드로이드/아이폰 각각 해시 워프 동작 체크
- Lighthouse로 퍼포먼스/접근성 기본 점검 후 UI 대비 색상 개선
