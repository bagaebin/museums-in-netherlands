# 스타일 가이드

## 컬러 팔레트
- 배경: `#0f1115`
- 카드/사물함 면: `#1b1f27`, 그라데이션 보조 `#262b38`
- 포커스/악센트: `#c6ff00`
- 텍스트 메인: `#f7f8fa`
- 보조 텍스트: `#9aa0ab`
- 스트로크/보더: `rgba(255,255,255,0.08)`

## 타이포그래피
- 기본 폰트: `Inter`, 시스템 폰트 폴백.
- 제목: 24~32px, 700~800 weight.
- 본문: 14~16px, 400~500 weight.
- 캡션/라벨: 11~12px, 500 weight, `letter-spacing: 0.4px` 권장.

## 사물함 시각 규칙
- 코너 라운드: 16px
- 스트로크 두께: 1px(도어), 관계 곡선 3px
- 그림자: `0 10px 30px rgba(0,0,0,0.35)`
- 도어 힌지 연출: `transform-origin: left center` 적용
- 내부 배경 확장 영역도 동일한 라운드 값 유지
